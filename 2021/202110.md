### 배운것

- html
  - 브라우저는 인간들이 쓰는 언어를 이해못한다.
  - 우리가 해야할일은 브라우저에게 콘텐츠가 무엇인지 알려줘야한다.
  - 마크업 언어
  - 브라우저에게 웹사이트의 구성이 어떻게 되어있는지 설명
  - html은 오류가 없다
- css
  - 디자인과 스타일을 위한 언어
  - 브라우저에게 웹사이트가 어떻게 보여야하는지에 대해 알려줌
- JavaScript
  - 웹 사이트를 동적으로 만들어주는 프로그래밍 언어
